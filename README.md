## 📖 Overview

Book Store is a full-stack CRUD web application that allows users to manage a collection of books. Users can create, read, update, and delete book records through a clean and responsive user interface.

This project demonstrates full-stack development using MongoDB, Express.js, React.js, and Node.js while following a scalable project structure. MERN CRUD applications are a common way to learn full-stack concepts and REST APIs. :contentReference[oaicite:0]{index=0}

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/samarthunhale-alt/Book-Store.git
```

Go inside the project

```bash
cd Book-Store
```

---

## Backend Setup

```bash
cd backend

npm install

npm run dev
```

---

## Frontend Setup

```bash
cd frontend

npm install

npm run dev
```

---

# ⚙ Environment Variables

Create a `.env` file inside the backend folder.

```env
PORT=5555

MONGO_URL=your_mongodb_connection_string
```

Example

```env
PORT=5555

MONGO_URL=mongodb://127.0.0.1:27017/book-store
```

# API Endpoints

| Method | Endpoint | Description |
|---------|----------|-------------|
| GET | /books | Get All Books |
| GET | /books/:id | Get Single Book |
| POST | /books | Create Book |
| PUT | /books/:id | Update Book |
| DELETE | /books/:id | Delete Book |
